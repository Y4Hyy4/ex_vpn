# Ex_vpn

华科网安实验记录

## 需求分析
### server
- [ ] 加密隧道通信
  - [ ] TLS
- [ ] 支持多客户端
  - [ ] 主进程可以不断fork出新进程支持多客户端
  - [ ] 主进程可以从多个子进程接收数据 from pipe to tun
    - [ ] 子进程对应客户端id
    - [ ] 子进程对应客户端虚拟ip 向真实ip转换
  - [ ] 主进程发送数据需要判别 from tun to pipe
    - [ ] 子进程对应客户端真实ip 向虚拟ip转换
    - [ ] 子进程对应客户端id
    - [ ] 子进程对应pipe
- [ ] 易用性和稳定性
  - [ ] VPN客户端虚拟IP获取
  - [ ] VPN客户端虚拟IP配置
  - [ ] VPN客户端内网路由配置
  - [ ] 稳定性
